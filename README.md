# Phishing Email Detection Model 📧

A simple machine learning project using Scikit-learn to classify emails as **Phishing** or **Safe**.

## ✨ Features
- Train on a dataset of phishing and safe emails
- Convert email text into numbers using TF-IDF
- Use Naive Bayes classifier for detection
- Show accuracy and confusion matrix

## 📦 Requirements
- Python 3.x
